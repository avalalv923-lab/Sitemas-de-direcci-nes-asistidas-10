PRESENTACIÓN

Datos del Proyecto
Título descriptivo: Diagnóstico Autónomo y Puesta a Punto de Sistemas de Dirección Asistida.
Centro de interés / Temática: Mantenimiento de sistemas de dirección y uso de tecnología de diagnosis avanzada.
Nivel al que va dirigido: Ciclo Formativo de Grado Medio en Electromecánica de Vehículos Automóviles.
Temporalización aproximada: 8 horas (4 sesiones de 2 horas).
Materias implicadas: Sistemas de Transmisión de Fuerzas y Trenes de Rodaje / Formación y Orientación Laboral (en la parte de seguridad).
Idioma: Castellano.


Descripción General
El alumnado deberá enfrentarse a un vehículo con una avería real o simulada en el sistema de dirección (hidráulica o eléctrica). El objetivo no es solo la reparación física, sino el proceso de diagnóstico digital.

Aprenden a:

Pinche en la imagen para dirigirse al Genially.
- Utilizar software de diagnosis para identificar códigos de error (DTC).
- Interpretar esquemas técnicos digitales de forma independiente.
- Realizar el ajuste mecánico y el purgado/calibrado del sistema.
- Generar un Informe Digital de Conformidad que certifique que el vehículo es seguro para la circulación.


Justificación:
Planificación Doble (Pedagógica y Técnica): La SdA no se limita a "arreglar la dirección" (técnico), sino que estructura el aprendizaje para que el alumno entienda el porqué de cada paso y cómo el software de diagnosis se integra en el flujo de trabajo (pedagógico).
Autonomía Digital en el Diagnóstico: Para evitar la dependencia constante del docente, se incluye el uso de manuales de taller digitalizados y foros técnicos. El alumno debe aprender a resolver incidencias de comunicación entre la ECU y el equipo de diagnosis de forma autónoma.
Conciencia Ética y Seguridad: Dado que la dirección es un elemento crítico de seguridad, el proyecto hace hincapié en la responsabilidad ética del mecánico. No se trata solo de apagar un testigo en el cuadro, sino de asegurar la integridad del conductor, siguiendo protocolos estrictos de verificación.
